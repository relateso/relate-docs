### Customizable note templates

You can now create your own note templates or choose from pre-built templates based on established sales methodologies. These templates provide an organized structure for your sales meeting notes, ensuring that you capture important information consistently.  

### Built-in templates

You can build your own templates or use one of the prebuilt ones:
- BANT
- MEDDIC
- Relate Discovery Method

## Improvements and Fixes  
- Improved: Zapier 3.7.0 -- added Find/Create/Update deal actions and Zapier now displays detailed error messages when record creation or modification fails  
- Improved: Relate now disallows replying/forwarding when the thread owner (to, cc, bcc) is not present in the email.  
- Improved: Changed the organization detail page's org website URL section to be in-place editable.  
- Fixed: Fixed an issue where cc recipients not registered as contacts were omitted in "reply all."  
- Fixed: Fixed an issue where cc recipients were not properly set when replying to a cc'd email.  
- Fixed: Fixed an issue where it was not possible to send the email during draft auto-save.  
- Fixed: Fixed an issue where email templates could not be created/edited/used on the inbox page.  
- Fixed: Fixed an issue where the number of deals filter was reset when saved in the data view.
