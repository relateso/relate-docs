Notable additions and improvements to the Relate guide docs. For the full product changelog, see the [Changelog](https://www.relate.so/changelog/).

<Update label="2026-07-02" tags={["Improvement"]}>
  ## Improved the email sync guide

  Restructured [Email sync](/setup/email-sync) by connection method (Gmail · Outlook · IMAP), refreshed the connection screenshots, and consolidated the setup notes and FAQs.
</Update>

<Update label="2026-06-29" tags={["Feature"]}>
  ## Added booking-link prefill guidance

  Documented how to prefill the booking form via query parameters on the [Meeting Scheduler](/features/calendar) page — covering built-in fields and custom Advanced fields (by identifier), plus the option to lock prefilled values as read-only.
</Update>

<Update label="2026-06-18" tags={["Integration"]}>
  ## Added the Google Contacts sync guide

  Added [Google Contacts Sync](/setup/google-contacts) — connect your Google account to automatically import your Google Contacts into Relate. Introduced a new Importing Contacts section.
</Update>

<Update label="2026-06-16" tags={["API"]}>
  ## Added the Relate Public API reference

  Introduced an API section with the [API reference](/api/introduction) — covering authentication, pagination, and endpoints for Contacts, Organizations, Notes, Custom Fields, Lists/Processes, Entries, Calendar, and Recordings.
</Update>

<Update label="2026-06-15" tags={["AI"]}>
  ## Added the Relate MCP guide

  Introduced a new AI section with the [Relate MCP](/ai/relate-mcp) doc — how to connect AI agents like Claude Code and Codex to Relate, what you can do with MCP, and example prompts.
</Update>

<Update label="2026-01-21" tags={["Feature"]}>
  ## Process & List docs, import copy

  Added Process and List guides, with renaming and copy improvements across the import docs.
</Update>

<Update label="2026-01-16" tags={["Feature"]}>
  ## AI Import docs update

  Updated the AI Import documentation.
</Update>

<Update label="2025-07-31" tags={["Feature"]}>
  ## List export, form responses, campaigns billing

  Added guides for exporting lists, downloading form responses, and the campaigns billing policy.
</Update>
